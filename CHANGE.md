# 更新日志

## v1.0.0 (2023-04-03)

### 新特性
- 基于Vue 3开发的组件库，包含13个常用组件
- 支持按需引入和全量引入
- 支持TypeScript类型定义
- 提供完整的API文档和示例
- 支持CDN引入和NPM安装

### 组件列表
- Button 按钮
- Input 输入框
- Icon 图标
- Checkbox 复选框
- Radio 单选框
- Toggle 开关
- Alert 警告提示
- Tooltip 提示框
- Select 选择器
- Dropdown 下拉菜单
- Tab 标签页
- Slider 滑块
- Message 消息提示

### 技术栈
- Vue 3
- Vite
- TypeScript (类型定义)
- ESLint
- Babel
