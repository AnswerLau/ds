设计系统 - CDN使用示例

通过CDN引入使用Vue组件库，快速构建界面

Button 按钮

基础用法
默认按钮 主要按钮 次要按钮 文本按钮 强调按钮
带图标按钮 成功操作 禁用按钮 加载中 
<ds-button>默认按钮</ds-button>
<ds-button type="primary">主要按钮</ds-button>
<ds-button type="secondary">次要按钮</ds-button>
<ds-button icon-left="home">带图标按钮</ds-button>
<ds-button disabled>禁用按钮</ds-button>

Icon 图标

基础图标
<ds-icon name="home"></ds-icon>
<ds-icon name="user"></ds-icon>
<ds-icon name="settings"></ds-icon>

Checkbox 复选框

基础用法
选项1 选项2

选中状态: 选项1 - {{ checked1 ? '是' : '否' }}, 选项2 - {{ checked2 ? '是' : '否' }}

<ds-checkbox v-model="checked1">选项1</ds-checkbox>
<ds-checkbox v-model="checked2">选项2</ds-checkbox>

Radio 单选框

基础用法
选项1 选项2

当前选中: {{ radio }}

<ds-radio v-model="radio" label="1">选项1</ds-radio>
<ds-radio v-model="radio" label="2">选项2</ds-radio>

Toggle 开关

基础用法

开关状态: {{ toggleValue ? '开启' : '关闭' }}

<ds-toggle v-model="toggleValue"></ds-toggle>

Input 输入框

基础用法

输入内容: {{ inputValue }}

<ds-input v-model="inputValue" placeholder="请输入内容"></ds-input>
<ds-input placeholder="带清除图标" clearable></ds-input>
<ds-input prefix="user" placeholder="带前缀图标"></ds-input>

Message 消息

基础用法
信息提示 成功提示 警告提示 错误提示 
// 在方法中使用
methods: {
  showInfo() {
    this.$message.info('这是一条信息提示');
  },
  showSuccess() {
    this.$message.success('这是一条成功提示');
  }
}
